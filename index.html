HTML y Python

Esta pagina explica las etiquetas de HTML y codigo de Python

=1. HTML

HTML (siglas en inglés de "Lenguaje de marcas de hipertexto") es el lenguaje de programación en el que están escritas las páginas web. Este lenguaje de codificación se utiliza en Internet. Los navegadores web leen y traducen el código HTML para mostrar el contenido de las páginas.
